# ⚽ LOAD EXTENSION IN CHROME NOW - EXACT STEPS

## 🎬 Deploy to Chrome in 5 Steps

### Step 1: Open Chrome Extensions Page
```
1. Open Chrome browser
2. Type in address bar: chrome://extensions/
3. Press Enter
```

### Step 2: Enable Developer Mode
```
1. Look at TOP RIGHT corner
2. Click toggle: "Developer mode"
3. It turns blue/ON
```

### Step 3: Click "Load unpacked"
```
1. Look at TOP LEFT area
2. Click blue button: "Load unpacked"
```

### Step 4: Select Project Folder
```
1. Navigate to: C:\Users\matab\Documents\bot pari
2. Select folder: football-predictor-clean
3. Click: "Select Folder"
```

### Step 5: Extension Loads!
```
1. Extension appears in chrome://extensions/
2. You see a small icon in top right corner
3. Blue "⚽ FOOTBALL AI" icon shows up
```
